# Памятка по DACH B2B (Германия / Австрия / Швейцария)

Когда использовать. При работе с B2B клиентами в DACH регионе - SaaS, professional services, manufacturing.

## Основной канал

LinkedIn 60-95% бюджета, не Meta. Meta идёт как retargeting / brand awareness 5-15%.

В DACH B2B профессионалы живут в LinkedIn. Meta для них - личное, не работа. Email из Meta-форм даёт низкое качество leads.

## Особенности LinkedIn в DACH

— Document Ads > Video Ads для legal / consulting / wealth (long-form lead magnets)
— Conversation Ads хорошо работают для outreach в Sponsored InMail
— Industry / Job Title / Company Size — основные фильтры таргетинга
— Lookalike не работает как в Meta — используй Account-Based Marketing (ABM)
— LinkedIn Insight Tag = аналог Meta Pixel, нужен на сайте

## 3-link decision chain

В DACH B2B решение принимают 3 роли:
— Initiator (тот кто ищет решение) — обычно operational manager / specialist
— Validator (тот кто проверяет) — обычно senior manager / department head
— Signatory (тот кто подписывает) — обычно C-level / partner / owner

Контент должен покрыть все 3 уровня:
— Initiator получает «как это работает» content
— Validator получает case studies / ROI calculations
— Signatory получает credibility signals / risk-mitigation content

## Compliance

— FINMA (Швейцария) для finance services
— BaFin (Германия) для banking / investments
— FADP (Швейцария, не GDPR)
— Datenschutzgrundverordnung (DE GDPR enforcement) - особенно строго
— Двойной opt-in для email обязателен
— Cookie Consent Mode v2 с 2024

## Языковая стратегия

— English для startups / scaleups / international companies
— German для Mittelstand (традиционный средний бизнес) - обязательно, иначе CR -40%
— Mix EN+DE landing pages для cross-segment campaigns

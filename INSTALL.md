# Установка ai-ops-7142

Это пошаговая инструкция как поставить плагин и проверить что он работает. Выбери свою ветку - Cowork (приложение Claude на Маке или Винде) или Claude Code (терминал).

Если ни того ни другого ещё нет - смотри последний раздел.

---

## Ветка А - у тебя Cowork (десктоп Claude)

Cowork - это приложение Claude которое скачивается на компьютер. Если ты открываешь Claude через значок программы на Маке или Винде (а не через сайт claude.ai в браузере) - это оно.

### Шаг 1. Открой новый чат

Запусти Cowork. Создай новый чат (кнопка `+` слева).

### Шаг 2. Введи первую команду

В поле для сообщения напиши - и отправь:

```
/plugin marketplace add OWNER/ai-ops-7142
```

`OWNER` - замени на имя владельца репозитория (Никита даст ссылку на курсе, нужно то что стоит до `/ai-ops-7142` в адресе репо).

Claude ответит что маркетплейс добавлен.

### Шаг 3. Введи вторую команду

Сразу после этого напиши - и отправь:

```
/plugin install ai-ops-7142@ai-ops-7142
```

Claude ответит что плагин установлен. Готово.

### Шаг 4. Проверь что скилы работают

Напиши Claude в обычном чате (без всяких слешей):

```
придумай 5 хуков для крео про английский для мам
```

Если Claude сразу выдаёт 5 готовых хуков в стиле копирайтера (короткие фразы, без воды, без «представь как ты», без штампов) - значит скил `ru-copywriter` подхватился и плагин стоит как надо.

Если Claude отвечает общими словами или предлагает «давай уточним аудиторию» - скилы не подхватились. Проверь что обе команды на шагах 2 и 3 прошли без ошибки.

## Ветка Б - у тебя Claude Code (терминал)

Claude Code - это режим Claude в терминале. Если ты запускаешь Claude командой `claude` в Терминале или iTerm - это он.

### Шаг 1. Открой Claude Code

В Терминале набери:

```
claude
```

Откроется чат внутри терминала.

### Шаг 2. Введи первую команду

В чате набери:

```
/plugin marketplace add OWNER/ai-ops-7142
```

`OWNER` - замени на имя владельца репозитория (Никита даст ссылку).

Подожди ответа - Claude напишет что маркетплейс добавлен.

### Шаг 3. Введи вторую команду

Сразу после:

```
/plugin install ai-ops-7142@ai-ops-7142
```

Claude напишет что плагин установлен. Готово.

### Шаг 4. Проверь

В том же чате напиши:

```
придумай 5 хуков для крео про английский для мам
```

Если получаешь готовые хуки в стиле копирайтера - всё работает. Если общая вода - скилы не встали, проверь команды на шагах 2 и 3.

## Если ничего нет - постановка Cowork

Cowork ставится один раз и работает дальше как обычное приложение.

1. Зайди на claude.ai/desktop с компьютера.
2. Скачай версию под свою систему - Mac или Windows.
3. Установи и залогинься в свой аккаунт Claude (тот же что в браузере).

Дальше открой Cowork и иди в раздел «Ветка А» этой инструкции.

---

## Что делать если не получилось

- Команда `/plugin` не распознаётся - проверь что у тебя свежая версия Cowork или Claude Code. Старые версии не знают про плагины.
- При установке ошибка `repository not found` - проверь правильно ли заменил `OWNER` на имя владельца репо. Должно быть точное имя как на GitHub.
- Команды прошли, но скилы не срабатывают на тестовую фразу - перезапусти Claude (закрой и открой приложение или терминал заново) и попробуй ещё раз.
- Ничего из этого не помогло - напиши Никите в чат курса со скриншотом ошибки.
